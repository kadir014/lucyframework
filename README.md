# lucyframework
My personal, [pygbag](https://github.com/pygame-web/pygbag)-friendly, toy framework built for Pygame.



# Installation
It's not on PyPI yet, you have to install yourself.

Clone the repo and `cd` into it.
```shell
$ git clone https://github.com/kadir014/lucyframework.git
```

If you are using `uv`, it should handle everything and run the example.
```shell
$ uv run lucyframework_example
```

If you don't want to use `uv`, you can install the framework manually.
```shell
$ python -m pip install .
```

Then you can run the example app to test things out.
```shell
$ python -m lucyframework_example
```
